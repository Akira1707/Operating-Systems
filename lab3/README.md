# Parallel Data Structures - Variant 14

**Structures implemented:**
- Fine-Grained Set (#2)
- MCSP Queue (#6)

**Compilation:**
To compile the project, simply run:
make

**Usage**
The program supports multiple test scenarios and benchmarking options. The general usage is:
./lab3 [OPTIONS]

Options:
--test <all|set_all|queue_all|set_bench|queue_bench>
--producers P (Number of producer threads)
--consumers C (Number of consumer threads)
--n N (Number of elements to process in the tests)
--cap CAP (queue capacity)
--runs R (Number of elements to process in the tests)
